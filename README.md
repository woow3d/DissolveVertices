# Dissolve 2-Edge Vertices

![Preview](https://github.com/woow3d/DissolveVertices/blob/main/imegas/view.jpg)

A Blender add-on that dissolves **selected vertices** connected to exactly two edges, helping clean up mesh topology efficiently.

## Features
- Dissolves selected vertices connected to exactly two edges
- Fast and lightweight
- Undo support
- Integrated into the Vertex menu
- No settings required

## Compatibility
- Blender 4.0+
- Edit Mode
- Mesh objects
- Windows / Linux / macOS

## Installation
1. Download the `.py` file
2. Open Blender
3. Edit → Preferences → Add-ons
4. Install…
5. Enable the add-on

## Usage
1. Select a Mesh object
2. Enter Edit Mode
3. Select vertices
4. Vertex → Dissolve 2-Edge Vertices

## Location
Edit Mode → Vertex → Dissolve 2-Edge Vertices
