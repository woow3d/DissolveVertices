# Dissolve 2-Edge Vertices

![Preview](iamegas/view.jpg)

إضافة لبرنامج Blender تقوم بحذف (Dissolve) **الرؤوس المحددة فقط** المتصلة بحافتين، للمساعدة في تنظيف التوبولوجي وتحسين بنية المجسم.

## المميزات
- حذف الرؤوس المحددة المتصلة بحافتين فقط
- سريعة وخفيفة
- تدعم التراجع (Undo)
- مدمجة في قائمة Vertex
- بدون إعدادات إضافية

## التوافق
- Blender 4.0+
- Edit Mode
- كائنات Mesh
- Windows / Linux / macOS

## التثبيت
1. تحميل ملف الإضافة `.py`
2. فتح Blender
3. Edit → Preferences → Add-ons
4. Install…
5. تفعيل الإضافة

## طريقة الاستخدام
1. اختر مجسم Mesh
2. انتقل إلى Edit Mode
3. حدّد الرؤوس
4. Vertex → Dissolve 2-Edge Vertices

## مكان الأداة
Edit Mode → Vertex → Dissolve 2-Edge Vertices
